A clean, high-visibility 64× resource pack optimized for PvP in Minecraft 26.1.2 and newer.

## Highlights

- 64× textures tuned for clarity and responsiveness in combat
- Minimal visual clutter for better target acquisition
- Ready to use in vanilla or modded clients

## Requirements

- Minecraft 26.1.2 or newer (pack format 84)

## Installation

1. Download the ZIP from the [latest release](https://github.com/honeyberries/pepmon270-64x-pvp-pack/releases/latest).
2. Launch Minecraft.
3. Options → Resource Packs → Open Pack Folder.
4. Move the downloaded `.zip` into the opened folder.
5. Back in Minecraft, select the pack in the Available list and move it to Selected. Click Done.
